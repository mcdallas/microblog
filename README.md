## Usage

### Clone the repo
```
git clone https://github.com/mcdallas/microblog
cd microblog
```

Install requirements
```
pip install -r requirements.txt
```

or if you are using pipenv

```
pipenv sync
```

### Initialize the database
```
python init_db.py
```

### Run run.py

```
python run.py
```

### Visit http://127.0.0.1:7000

### Run tests

```
pip install pytest
pytest
```
